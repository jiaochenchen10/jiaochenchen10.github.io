Install Ubuntu system dependecies

The purpose of this step is to prepare our system with the dependencies necessary for OpenCV.
To start, open up our command line and update the apt-get package manager to refresh and upgrade and preinstalled packages or libraries.
apt-get update doesn’t actually install new versions of software. Instead, it updates the package lists for upgrades for packages that need upgrading, as well as new packages that have just come to the repositories.
它更新的是需要即将被更新的包的信息列表。
apt-get upgrade will fetch new versions of packages existing on the machine if APT knows about these newversion by way of apt-get update.
apt-get dist-upgrade will do the same job which is done by apt-get upgrade, plus it will also intelligently handle the dependencies, so it might remove obsolete packages or add new ones.
假如家里要购买新的生活用具，apt-get update就好比对需要购买的用品列了一份清单，apt-get upgrade就是拿着这份清单去超市购买，apt-get dist-upgrade不仅会把新的买回来，还会把破旧的用具扔掉。

We’ll also need to install some developer tools as well as prerequisites required for image and video I/O, optimization, and creating plots/visualizations.

We’ll wrap up Step #1 by installing the Python development headers and libraries for Python 3.5.
`
$ sudo apt-get install python3-dev
